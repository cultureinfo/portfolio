girls who code projects

octorescue

octorescue is a simple scratch game. it is a sidescroller game, like games like flappy bird and geometry dash, just better. players have called it fun and addictive and a great game for all ages. although it is simple in code and has not been fully completed, the game is still in the works and will likely be completed. visit octorescue.

justeathealthy

just eat healthy is a girls who code club project, created by me and other girls in the montclair learning center girls who code club. the website has been finished and includes a game and video as well as basic information about the different food groups to teach younger girls how to choose nutritious foods. and of course, what is a good health site without recipes?! recipes include raspberry waffles, a banana and lemon smoothie, and even potato casserole! visit just eat healthy.
